# Lead
Lead can be found either in the bunker itself,or on <button topic="Toxic fields:ToxicField">Toxic fields</button>.
Lead can be put into lead refinery with either cobalt or adamantium to create metal alloys.
Adamantium-lead alloy is mostly used to upgrade weapons at lead forge.
Cobalt-lead alloy is mostly used to upgrade armour.
Both of them is used for lead-mixed pickaxe and bunker shielding block

# Bunker-shielding block 
Reinforced smooth vulcanic bricks,which can only be broken by lead-mixed pickaxe.
Recipe:20 blocks for 1 adamantium-lead alloy and 20 smooth vulcanic bricks.
